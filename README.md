# Currency Converter

A simple and modern **Currency Converter** web application built using **HTML, CSS, and JavaScript**.  
It fetches real-time exchange rates and allows users to convert between different currencies instantly.

---

## Features

- Convert between multiple world currencies
- Real-time exchange rates using an external API
- Clean, responsive UI with modern styling
- Input validation for user-friendly experience
- Lightweight and fast — no frameworks required

---

## Technologies Used

- **HTML5** – Structure
- **CSS3** – Styling & layout
- **JavaScript (ES6)** – Logic & API handling
- **ExchangeRate API** – Live currency data
